# MLX90640-With-STM32
MLX90640 is a fully calibrated 32x24 pixels thermal IR array sensor.

## MLX90640 with STM32 SWI2C 
* SDA_PIN:PB7
* SCL_PIN:PB6

The data will printf on UASRT1 with 115200 buad rate like the blow picture!
